## рҹ“Ң Overview

**Smart Courier Manager** is a production-ready, scalable backend system built with **Spring Boot 3** for managing the full lifecycle of courier operations вҖ” from user registration and order creation to intelligent delivery assignment, real-time location tracking, and automated email notifications.

Designed with a clean layered architecture, JWT security, and extensible design patterns, this system is built to handle real-world courier logistics at scale.

---

| Feature | Description |
|---|---|
| рҹ”җ **JWT Authentication** | Stateless token-based auth with role-based access control |
| рҹ‘Ҙ **Multi-Role System** | Customer, Delivery Agent, and Admin roles |
| рҹ“Ұ **Order Management** | Full CRUD with real-time status tracking |
| рҹҡҡ **Smart Assignment** | Single & bulk delivery assignment to agents |
| рҹ“Қ **Location Tracking** | Live location updates for in-transit deliveries |
| рҹ“§ **Email Notifications** | Automated Gmail SMTP email alerts |
| вҸ° **Scheduled Updates** | Cron-based automatic delivery status updates |
| рҹӣЎпёҸ **Global Error Handling** | Centralized exception management with structured error responses |
| вң… **Input Validation** | Jakarta Bean Validation on all request payloads |

## рҹҡҖ Quick Start

### Prerequisites

```bash
вң” Java 17+
вң” MySQL 8.0+
вң” Maven 3.8+
вң” Git
```

### 1пёҸвғЈ Clone the Repository

```bash
git clone https://github.com/HarshPatel-08/Smart-Courier-Manager.git
cd Smart-Courier-Manager
```

### 2пёҸвғЈ Setup Database

```sql
CREATE DATABASE courierManagement_db;
```

### 3пёҸвғЈ Configure Application Properties

```bash
cp src/main/resources/application.properties.example src/main/resources/application.properties
```

Edit `application.properties`:

```properties
# Database
spring.datasource.url=jdbc:mysql://localhost:3306/courierManagement_db?allowPublicKeyRetrieval=true&useSSL=false
spring.datasource.username=YOUR_DB_USERNAME
spring.datasource.password=YOUR_DB_PASSWORD

# JWT
jwt.secret=your-super-secret-key-at-least-32-characters-long
jwt.expiration=86400000

# Gmail SMTP (use App Password, not your real password)
spring.mail.username=your_email@gmail.com
spring.mail.password=your_gmail_app_password
```

> рҹ’Ў **Gmail App Password**: Go to `Google Account вҶ’ Security вҶ’ 2-Step Verification вҶ’ App Passwords`

### 4пёҸвғЈ Run the Application

```bash
# Using Maven Wrapper (recommended)
./mvnw spring-boot:run

# Or on Windows
mvnw.cmd spring-boot:run
```

```
Server starts at вҶ’ http://localhost:8080
```

### рҹ“Ұ Orders

> All order endpoints require: `Authorization: Bearer <token>`

| Method | Endpoint | Description | Role |
|--------|----------|-------------|------|
| `POST` | `/order/create` | Create a new order | Customer |
| `GET` | `/order/{orderId}` | Get order by ID | Any |
| `GET` | `/order/my-orders` | Get current user's orders | Customer |
| `GET` | `/order/all` | Get all orders | Admin |

## рҹ”җ Security

- **JWT (JSON Web Token)** вҖ” stateless authentication via `io.jsonwebtoken` (JJWT 0.12.3)
- All protected routes validated via `JwtAuthenticationFilter` on every request
